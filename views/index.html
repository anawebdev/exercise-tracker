Exercise tracker

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

GET users's exercise log: GET /api/exercise/[log]/[userId]/{&from}/{&to}/{&limit}

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Examples

Get the all the exercise info in .json format of user_1: /api/exercise/log/59944f5afa812a3440670d06

Get the all the exercise info belonging to user_1, from 3rd of October until 15th of October 2017: /api/exercise/log/59944f5afa812a3440670d06/2017-10-03/2017-10-15

Get the first 3 instances of exercise info belonging to user_1, from 3rd of October until 15th of October 2017: /api/exercise/log/59944f5afa812a3440670d06/2017-09-03/2017-10-20/3
